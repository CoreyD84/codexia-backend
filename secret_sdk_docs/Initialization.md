# GlobexTracker Initialization

Before using the `GlobexTracker`, it must be initialized with a valid API key.

## `init(apiKey: String)`

This method should be called once when the application starts.

- **apiKey**: Your unique API key for the GlobexTracker service.

**Example (Kotlin):**
```kotlin
GlobexTracker.init("YOUR_API_KEY")
```
